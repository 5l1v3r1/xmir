# xmir
web server fingerprinting


## supported servers

- [x] Joomla
- [x] WordPress
- [ ] WebLogic
- [ ] Struts2
- [ ] MS IIS

## usage

```
Usage of ./xmir:
  -file string
    	Masscan result file in XML format
  -outfile string
    	IP list file for XML parser output
  -result string
    	Scan result
  -server string
    	Specify a server software
```
